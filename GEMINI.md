# Management Consultant Skill

This repository is an MBB-level (McKinsey / Bain / BCG) management-consulting skill: structured problem solving (MECE issue trees, hypothesis-driven analysis), strategy frameworks, market sizing / guesstimation, case-interview structures, industry reference libraries, and executive-grade deliverable templates.

**To use it:** load the full skill definition at [`skill/SKILL.md`](skill/SKILL.md). It routes to a deep reference library under `skill/references/` and worked examples under `skill/case-studies/` — open those files on demand as the skill instructs.

The content is model-agnostic: it works in any capable LLM. Agents that read `AGENTS.md` (Codex, Cursor, Windsurf, Amp, and others) will pick this up automatically.
